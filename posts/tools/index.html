一、常用工具

Idea 插件

Free MyBatis Plugin

  • 生成mapper xml文件
  • 快速从代码跳转到mapper及从mapper返回代码
  • mybatis自动补全及语法错误提示

GenerateAllSetter

alt+enter快捷键，一键生成所有setter 方法。

camelCase

使用 快捷键 shift + alt + u ，变量就会改变，支持大小写，下划线，驼峰，首字母大写等转化。

sonarLint /Alibaba Java Coding Guidelines

代码质量检查工具.

RESTfultoolkit

通过快捷键ctrl +\来进行搜索路径。

Chrome 插件

JSON-handle

快速格式化并查看Json 字符串

截图工具
